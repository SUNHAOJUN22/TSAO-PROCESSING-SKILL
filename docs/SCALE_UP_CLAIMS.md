# Scale-up claims

Scale-up is a set of falsifiable physics claims, not a capacity ratio.

Each claim records:

- source and target scale;
- equipment geometry and operating mode;
- conserved and non-conserved dimensionless groups;
- residence-time, mixing, heat-transfer and mass-transfer comparisons;
- phase regime, viscosity, solids or polymer architecture state;
- measurement equivalence and sampling boundary;
- model applicability and uncertainty;
- broken similarity and compensating pilot evidence;
- acceptance criterion, owner, reviewer and Gate impact.

A pilot may be geometrically smaller yet dynamically representative, or geometrically similar yet scientifically unrepresentative. TSAO requires explicit proof of the variables that control the decision.
