# Thermodynamic and physical-property method qualification

Method selection begins with the decision and expected phase/association/electrolyte/polymer regime, not a favourite simulator package. Record components, pseudo-components, reference states, interaction parameters, data sources, valid temperature/pressure/composition ranges and critical properties.

Compare predictions with independent VLE/LLE/SLE, density, enthalpy, heat capacity, viscosity, diffusion and phase-stability evidence as applicable. Screen limiting behaviour and derivative continuity. Polymer solutions, electrolytes, associating systems and supercritical conditions require methods suitable for their physics.

A property package that closes a flowsheet numerically is not qualified by convergence alone.
