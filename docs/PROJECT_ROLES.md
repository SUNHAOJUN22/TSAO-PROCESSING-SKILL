# Project roles

Typical accountable roles include sponsor, technology owner, project manager, chemistry/catalyst lead, analytical lead, thermodynamics lead, kinetic/reactor lead, separation/recycle lead, scale-up/pilot lead, controls lead, process-safety lead, reliability lead, TEA/LCA lead, IP counsel, customer/application lead, data/model governance lead and independent reviewer.

A role may be performed by the same person in low-consequence work, but authorship, approval and independent review cannot be conflated where Gate or model-risk rules require separation.
