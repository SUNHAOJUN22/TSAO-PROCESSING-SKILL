# Human accountability

TSAO automates research organization, data-contract enforcement, calculations, document generation and software verification. It does not eliminate accountable professional judgment.

Named humans are required for research-question ownership, experimental authorization, data acceptance, model-risk review, process-safety decisions, legal FTO, customer qualification, commissioning and industrial performance guarantees. High-consequence reviewers must be sufficiently independent of the original author or model developer.

The system should ask for human decisions only when they materially change project correctness, safety, legal posture or resource commitment. Routine artifact generation and deterministic checks should proceed automatically.
