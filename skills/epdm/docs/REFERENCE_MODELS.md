# EPDM P1 reference models

The P1 layer supplies minimum transparent calculations for active-site fraction, E/P/diene insertion fractions, transfer/deactivation-limited degree of polymerization, retained unsaturation, branching and gel-risk indices, heat-removal margin, mixing Reynolds number, recycle-poison steady state, devolatilization residual, grade-transition off-spec fraction and a Mooney trend reference.

Every output is `CALCULATED_REFERENCE_ONLY`. Parameter fitting, multi-site distributions, uncertainty propagation, commercial simulator recovery, reactor relief design and customer performance remain external work.
