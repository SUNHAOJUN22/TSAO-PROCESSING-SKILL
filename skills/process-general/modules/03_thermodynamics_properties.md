# Thermodynamics and physical properties

**Decision:** Which property model and data are adequate for the chemistry, phase regime and design decision?

**Gate interface:** G5–G13. Cover VLE/LLE/SLE, density, enthalpy, heat capacity, viscosity, diffusivity, association, electrolytes, solids and critical-region behaviour.

**Required work:** justify method selection; validate binaries/ternaries and limiting behaviour; compare an independent method/tool for high-consequence results; map interpolation/extrapolation, metastability, precipitation and fouling limits; propagate property uncertainty to equipment and control decisions.

**Fail closed:** HOLD when a simulator default is used without validation or the requested state lies outside the property model domain.

**Outputs:** property-method dossier, parameter/data lineage, validation plots, applicability map and uncertainty-sensitive design variables.
