# Universal process-package platform

`tsao.process_package` is the executable contract shared by petrochemical, polymer, fine/batch, bioprocess, electrochemical and solids routes.

It validates the design basis, component and stream identity, equipment references, composition closure, equipment mass/energy balance, utilities, controls, HSE safeguards, evidence qualification, acceptance criteria and named approvals. Structural defects return `FAIL`; unresolved scientific, HSE, control or approval work returns `HOLD`; only a fully evidenced package returns `PASS`.

This is an open-software content audit, not engineering approval. Relief design, HAZOP/LOPA/SIL, commercial simulator execution, equipment design, licensed property databases and plant performance remain external accountable work.
