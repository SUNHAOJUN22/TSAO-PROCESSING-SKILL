# Reports

This directory contains machine and human-readable software qualification, release, migration and project audit reports. Reports must state the exact source identity and distinguish artifact qualification from technical approval.

- [`LINEAGE_COMPLETENESS_AUDIT_ALPHA3.md`](LINEAGE_COMPLETENESS_AUDIT_ALPHA3.md) compares the earliest SJTU-POE/universal-polymer mission, EPDM v9 and TSAO alpha.2, records alpha.3 fixes and states remaining public-source parity gaps.
- `CODE_AUDIT_ALPHA2.md` records the alpha.2 code-hardening work.
- `QUALIFICATION_BOUNDARY.md` defines what repository CI can and cannot approve.
