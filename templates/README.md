# TSAO project templates

Project initialization may copy governance and engineering templates from this directory. Every populated template must retain provenance, version, owner, status, uncertainty and approval fields.

Recommended template families:

- charter, RACI and decision register;
- requirements, CQA, CMA and CPP;
- evidence, standards, patents and contradiction ledger;
- material, sample, method, dataset and model passports;
- experiment, continuous-bench, pilot and demonstration plans;
- balance, PFD, stream, equipment, utility and control records;
- scale-up claims, uncertainty, risks and change control;
- HSE, reliability, TEA, LCA, supply-chain and FTO handoffs;
- customer qualification, commissioning, performance testing and transfer.

An empty template is not evidence and never advances a Gate.
